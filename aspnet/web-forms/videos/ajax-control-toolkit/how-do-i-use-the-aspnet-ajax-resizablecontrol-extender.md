<a name="how-do-i-use-the-aspnet-ajax-resizablecontrol-extender"></a>[Como fazer:] Usar o extensor ResizableControl do ASP.NET AJAX?
====================
por [Joe Stagner](https://github.com/JoeStagner)

O extensor ResizableControl do Kit de ferramentas de controle AJAX ASP.NET permite que o desenvolvedor especifique uma região da página da Web que o usuário pode redimensionar dinamicamente. Este vídeo demonstra como configurar a aparência da região redimensionável e como executar o script do lado do cliente ao redimensionamento.
