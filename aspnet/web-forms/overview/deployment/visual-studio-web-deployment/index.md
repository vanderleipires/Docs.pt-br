<a name="aspnet-web-deployment-using-visual-studio"></a>Implantação de Web do ASP.NET usando o Visual Studio
====================

Requer o Visual Studio 2012 e o ASP.NET and Web Tools 2012.2 atualização ou o Visual Studio 2010.
